1. What is the difference between null and undefined?
   Answer: Null is defined as the intentional absence of a value and undefined termed as the uninitialized state of a variable or a variable that has not been assigned a value yet.

   5.Explain the concept of Scope in JavaScript (Global, Function, Block).
   Answer:
   Global Scope:Global scope refers to a variable that is declared outside any function or block, so it can be used anywhere in the program, both inside functions and in the main code.

Function Scope:Function scope refers to a variable that is only accessible within that function and cannot be used outside that function.

Block Scope:Block scope refers to variables that declared with let or const inside a { } block. These variables are accessible only within that block and it cannot be used outside.
